Roadmap Belajar IT

Langkah pertama: Cara Menggunakan GitHub dari Awal

1. Buat Akun GitHub

Kunjungi github.com, lalu klik Sign up dan ikuti instruksinya.

2. Install Git

Download dan install Git dari git-scm.com. Setelah itu, coba jalankan git --version di terminal.

3. Konfigurasi Git

Gunakan perintah:
git config --global user.name "Namamu"
git config --global user.email "email@example.com"

4. Buat Repository

Login ke GitHub & klik ‚ÄúNew repository‚Äù, beri nama dan deskripsi, lalu klik Create.

5. Clone Repository

Copy link repo & jalankan:
git clone https://github.com/username/nama-repo.git

6. Push & Pull

Simpan perubahan dengan:
git add .
git commit -m "Pesan"
git push origin main
Untuk update: git pull
